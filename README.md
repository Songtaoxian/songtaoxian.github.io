# Personal Academic Site

This is a static personal academic homepage inspired by the structure of an AcademicPages-style profile site.

## Pages

- `index.html`: research homepage
- `publications/`: publications
- `running/`: running log
- `blog/`: notes and posts
- `cv/`: CV

## Customize

Edit the text in the HTML files and replace:

- `Tianxiao Shao`
- `hello@example.com`
- GitHub and Google Scholar URLs
- publication/project titles
- `assets/img/avatar.svg`

## Publish With GitHub Pages

Create a repository named `<your-github-username>.github.io`, upload these files to the repository root, and enable GitHub Pages from the `main` branch root.
